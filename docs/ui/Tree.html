树 Tree（封装于ztree）

示例

//HTML
<div id="treeId"></div>

//JavaScript
var tree = new Tree('treeId', option);

//Option
option = {
    url: String 数据请求url,
    data: Array 初始化列表结构数据,
    simpleData: bool 是否是列表数据，默认true,
    check: bool 是否显示复选框,
    onLoad: Function 初始化加载回调函数，参数data:Array,
    onClick: Function 点击节点回调函数，参数node:Object
}

属性

名称 类型 描述
id String 控件id。
list Array 原始列表数据。
data Array 树状结构数据。
selectedNode Object 选中的树节点对象。

方法

名称 参数 返回值 描述 示例
setData data：Array 设置树控件数据。
reload 重新加载树节点。
getCheckedNodes Array 获取勾选的节点对象数组。
getNodeData filter：Function Object，节点数据 根据条件函数获取原始列表数据对象。
