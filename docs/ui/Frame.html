全局方法

名称 参数 返回值 描述 示例
getCurTab Object 获取当前打开的标签页。
showTab url: String 显示指定url的标签页。
log obj: Object 日志打印。

工具类 Utils

名称 参数 返回值 描述 示例
getUrlParam name: String String 获取ur中的参数。 var name = Utils.getUrlParam('name');
addCss href: String 动态添加样式文件。 Utils.addCss('xxx.css');
addJs src: String 动态添加脚本文件。 Utils.addJs('xxx.js');
copy text: string 复制数据到剪切板。 Utils.copy('...');
paste enabled: bool
callback: Function		 粘贴剪切板数据。 Utils.paste(function(text){...});
list2Tree data, key, pid 将列表数据转成树结构。 Utils.list2Tree([], 'pid', '0');
tree2List data, key 将树结构转成列表数据。 var list = Utils.tree2List([], 'pid');
setUser user: Object 设置客户端用户信息。 Utils.setUser(user);
getUser Object 获取客户端用户信息。 var user = Utils.getUser();
setCodes data: Array 设置客户端缓存代码表。 Utils.setCodes([]);
getCodes category: String 根据代码类别获取客户端缓存的代码对象数组。 var codes = Utils.getCodes('Category');
genFile text, saveName 将字符串生成文件并下载。 Utils.genFile('....', 'test.txt');

WebSocket操作类 WsUtil
