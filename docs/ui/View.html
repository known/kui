属性

名称 类型 描述
name String 视图名称。
option Object 选项。

方法

名称 参数 返回值 描述 示例
loadGrid where: Object 加载视图网格数据。 view.loadGrid(where);
setGridData data: Array 设置视图网格数据。 view.setGridData(data);
getRowData filter: Function row/rows 根据条件函数获取行数据对象。 var row = view.getRowData(function(d){...});
