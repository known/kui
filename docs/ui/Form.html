表单 Form

示例

//HTML
<div id="formTest"></div>

//JavaScript
var grid = new Form('Test', option);

//Option
option = {
    url: String 数据请求url,
    data: Object 默认表单数据,
    title: String 标题,
    titleInfo: Function 标题格式化回调函数，参数data,
    toolbar: Array 按钮对象数组, {text,icon,handler},
    setData: Function 填充数据后回调函数，参数{form,data},
    onSaving: Function 保存前回调函数，参数data,
    submitData: Function 获取保存前附加提交的数据对象
}

属性

名称 类型 描述
id String 控件id。
title String 表单标题。
label Object 字段共用栏位对象。

方法

名称 参数 返回值 描述 示例
show data: Object 显示表单。
close 关闭表单。
validate bool 验证表单字段。
clear 清空表单字段。
getData 获取表单数据对象。
setData data: Object 设置表单数据。
setReadonly readonly: bool 设置表单只读或编辑模式。
load url: String
callback: Function 		加载表单数据。
save url: String
callback: Function 		保存表单。
