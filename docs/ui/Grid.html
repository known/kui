网格 Grid

示例

//HTML
<div class="grid">
    <table id="gridTest"></table>
</div>

//JavaScript
var grid = new Grid('Test', option);

//Option
option = {
    index: bool 显示行号，默认true,
    fixed: bool 是否铺满父容器，默认true,
    edit: bool 
    width: int/string
    columns: []
    url: string
    where: {}
    sortField: string
    sortOrder: string
    autoQuery: bool
    page: bool,
    toolbar: {}
    action: {},
    summary: function(d) {}
}

属性

名称 类型 描述
name String 控件name。
option Object 选项。
elem Object 表格jquery对象。
columns Array 取得表格栏位对象集合。
data Array 取得表格数据对象集合。
where Object 取得查询条件对象。

方法

名称 参数 返回值 描述 示例
setColumns cloumns: Array, 栏位对象集合。 设置网格栏位，可以实例化后设置（用于动态栏位），
也可以通过option参数实例化时设置。		 
grid.setColumns([{},{}]);
var grid = new Grid('Test', {
    columns: [{},{}]
});
reload where: Object, 查询参数对象。 重新加载网格数据。 grid.reload(where);
setData data: Array, 网格数据对象集合。 设置网格数据。 grid.setData([{},{}]);
getSelected Array 获取选择行的对象集合。 var rows = grid.getSelected();
getData Array 获取网格数据（包含编辑后的数据）。 var rows = grid.getData();
getValue row: Object
column: Object		 Object 获取网格单元格值。
addRow row: Object, 新行数据对象。 添加一行，只有编辑表格可用。 grid.addRow({});
removeRow index: int, 行序号。 删除指定行。 grid.removeRow(index);
moveRow index: int, 行序号。
direct: String, 移动方向，up/down。
url: String, 后端更新顺序url[可空]。 		移动网格行。 grid.moveRow(index, 'up');
