列表框 ListBox

示例

//HTML
<div id="lbTest"></div>

//JavaScript
var rbl = new ListBox(elem, option);
//elem: 'lbTest' 或 $('#lbTest')  控件id或对象。

//Option 选项可选配url和data初始化列表数据项
option = { 
    url: '', 
    data: [], 
    onClick: function (item) {} 
}

属性

名称 类型 描述
id String 控件id。取值
selected Object 选中的数据对象。取值

方法

名称 参数 返回值 描述 示例
setData data: Array 设置列表数据项。数据支持如下类型：
1.代码表数组；
2.简单数组；
3.客户端缓存代码类别。		 
lb.setData([
    { Code: 'test1', Name: '测试1', Data: { Type: 'test', Unit: 'U1' } },
    { Code: 'test2', Name: '测试2', Data: { Type: 'test', Unit: 'U2' } },
    { Code: 'test3', Name: '测试3', Data: { Type: 'test', Unit: 'U3' } }
]);

lb.setData(['项1','项2','项3','项4','项5']);
