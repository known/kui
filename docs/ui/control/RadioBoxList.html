单选框列表 RadioBoxList

示例

//HTML
<div id="rblTest"></div>

//JavaScript
var rbl = new RadioBoxList(elem, option);
//elem: 'rblTest' 或 $('#rblTest')  控件id或对象。

//Option 选项可选配url和data初始化列表数据项
option = {
    name: '',
    url: '', 
    data: [] 
}

属性

名称 类型 描述
name String 控件name。取值
data Array 列表绑定数据项对象数组。取值

方法

名称 参数 返回值 描述 示例
setData data: Array 设置列表数据项。数据支持如下类型：
1.代码表数组；
2.简单数组；
3.客户端缓存代码类别。		 
rbl.setData([
    { Code: 'test1', Name: '测试1' },
    { Code: 'test2', Name: '测试2' },
    { Code: 'test3', Name: '测试3' }
]);

rbl.setData(['项1','项2','项3','项4','项5']);
测试1 测试2
getValue String 获取控件值。 var value = rbl.getValue(); 测试
setValue value: String 设置控件值。 rbl.setValue('test1'); 测试
