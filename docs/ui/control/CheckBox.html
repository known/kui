复选框 CheckBox

示例

var cb = new CheckBox(elem, data); 
//elem: 'Enabled' 控件name或对象。
//data: {Code: 1, Name: '启用'}

属性

名称 类型 描述
name String 控件name。取值
data Object 控件绑定数据对象。取值
elem JqObject 控件dom对象。打印

方法

名称 参数 返回值 描述 示例
getValue String 获取控件值，选中返回值，否则返回0。 var value = cb.getValue(); 取值 赋正确值 赋错误值
setValue value: String 设置控件值，值包含绑定值选中，否则不选中。 cb.setValue(); 取值 赋正确值 赋错误值
