通用控件 Input

示例

var input = new Input(dom, option);

属性

名称 类型 描述
type String 输入框类型（html|editor|textarea|text|hidden|radio|checkbox）。
id String 输入框id属性。
name String 输入框name属性。
elem Object 输入框dom对象。

方法

名称 参数 返回值 描述 示例
setReadonly readonly: bool 设置只读输入框。
validate bool 验证输入框。
getValue 获取输入框值。
setValue value: string/Date 设置输入框值。
change callback：Function({selected}) 输入框值变化事件。
