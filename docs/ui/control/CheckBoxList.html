复选框列表 CheckBoxList

示例

//HTML
<div id="cblTest"></div>

//JavaScript
var cbl = new CheckBoxList(elem, option);
//elem: 'cblTest' 或 $('#cblTest')  控件id或对象。

//Option 选项可选配url和data初始化列表数据项
option = {
    name: '',
    url: '', 
    data: [] 
}

属性

名称 类型 描述
name String 控件name。取值
items Array 列表项控件对象数组。打印

方法

名称 参数 返回值 描述 示例
getCheckedData Array 获取选中的数据项对象数组。 var data = cbl.getCheckedData(); 测试
setData data: Array 设置列表数据项。数据支持如下类型：
1.代码表数组；
2.简单数组；
3.客户端缓存代码类别。		 
cbl.setData([
    { Code: 'test1', Name: '测试1' },
    { Code: 'test2', Name: '测试2' },
    { Code: 'test3', Name: '测试3' }
]);

cbl.setData(['新增','编辑','删除','导入','导出']);
测试1 测试2
getValue String 获取列表选中项的值。 var value = cbl.getValue(); 测试
setValue value: String 设置列表数据值。 cbl.setValue('test1,test3,新增,删除'); 测试
